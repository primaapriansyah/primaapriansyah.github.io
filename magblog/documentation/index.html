Head MarkUp

In head you will want to change favicon and title.

<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8"/>
		<meta name="viewport" content="width=device-width"/>
		<meta name="theme-color" content="#f20e0e"/>
		<meta name="msapplication-navbutton-color" content="#f20e0e"/>
		<meta name="apple-mobile-web-app-status-bar-style" content="black"/>
		<meta name="mobile-web-app-capable" content="yes"/>
		<meta name="apple-mobile-web-app-capable" content="yes"/>
		<link rel="icon" type="image/x-icon" href="images/favicon.png"/>
		<link rel="shortcut icon" type="image/x-icon" href="images/favicon.png"/>
		<link rel="apple-touch-icon-precomposed" href="images/favicon.png"/>
		<link rel="icon" type="image/png" href="images/favicon-192x192.png" sizes="192x192"/>
		<title>MagBlog - News Editorial & Magazine HTML5 Template</title>
		<link rel="stylesheet" href="css/grid.css" media="all"/>
		<link rel="stylesheet" href="css/framework.css" media="all"/>
		<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Overpass:400,400i,600,800" media="all"/> 
		<link rel="stylesheet" href="css/fontawesome.css" media="all"/>
		<link rel="stylesheet" href="css/normalize.css" media="all"/>
		<link rel="stylesheet" href="css/owl.carousel.css" media="all"/>
		<link rel="stylesheet" href="style.css" media="all"/>
		<link rel="stylesheet" href="css/customizer.css" media="all"/>
		<link rel="stylesheet" href="css/responsive.css" media="all"/>
	</head>
Heads up! The normalize.css is a generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.
Stylesheets

Here's a list of the stylesheet files I'm using with this template, you can find more information opening each file.

File Name Description
style.css The main CSS file of this template, Rules like headings and paragraphs, margins and body and more...
normalize.css Makes browsers render all elements more consistently and in line with modern standards.
framework.css Our custom codes help as to code quickly.
grid.css Is 12 Columns System Grid from Bootstrap.
customizer.css Rules for colors used as primary in site, like background, color and border color, transitions and more...
fontawesome.css Rules for font icons.
owl.carousel.css Rules for Carousel Sliders.
colorbox.css Rules for Images LightBox.
justifiedGallery.css Rules for Images Gallery.
responsive.css Rules used when you browsing with other devices, the template will be responsive on all screens.
jQuery's Files

Here's a list of the jQuery files I'm using with this template, you can find more information opening each file.

File Name Description
jquery.min.js Javascript UI library.
smoothscroll.js A jQuery plugin for smooth scrolling with the mouse wheel.
owl.carousel.min.js A jQuery plugin for Owl Carousel Slider.
theia-sticky-sidebar.js A jQuery plugin that make sidebar sticky when you scroll down.
matchheight.js A jQuery plugin to match height of grid posts.
unvil.js A jQuery plugin for Lazy Load Images.
stickyme.js A jQuery plugin for make Header fixed when scroll down.
justifiedGallery.js A jQuery plugin for Images Gallery.
colorbox.js A jQuery plugin for Images LightBox.
main.js All scripts are initialized from this file.
